Halaman Berita

Web media online berita sederhana menggunakan bootstrap dengan admin panel untuk manajemen berita dan halaman utama berita, berikut adalah halaman utama ketika pengguna masuk, berisikan daftar berita-berita.

Halaman Detail Berita

Halaman ketika pengguna masuk ke salah satu berita yang dipilih, menampilkan detail dari berita yang dipilih.

Halaman Login Admin

Halaman login untuk admin, dengan memasukkan email dan password untuk masuk ke admin panel.

Halaman Dashboard Admin

Halaman dashboard admin berisikan jumlah dari data kategori, data post dan data admin.

Halaman Kategori

Halaman kategori pada admin panel, berisikan kategori-kategori yang telah ditambahkan.

Halaman Post

Halaman post pada admin panel, berisikan post berita yang telah ditambahkan.

Halaman Tambah Post

Halaman tambah post untuk menambahkan post berita dengan memasukkan judul berita, gambar berita, isi berita, dan memilih kategori berita sesuai dengan kategori yang telah ditambahkan.

Halaman Feedback

Halaman feedback, menampilkan feedback yang diberikan oleh pengguna.

Halaman Home
