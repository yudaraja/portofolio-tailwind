Halaman Berita

Web media online berita sederhana menggunakan bootstrap dengan admin panel untuk manajemen berita dan halaman utama berita, berikut adalah halaman utama ketika pengguna masuk, berisikan daftar berita-berita.

Halaman Detail Berita

Halaman ketika pengguna masuk ke salah satu berita yang dipilih, menampilkan detail dari berita yang dipilih.

Halaman Login Admin

Halaman login untuk admin, dengan memasukkan email dan password untuk masuk ke admin panel.

Halaman Dashboard Admin

Halaman dashboard admin berisikan jumlah dari data kategori, data post dan data admin.

Halaman Kategori

Halaman kategori pada admin panel, berisikan kategori-kategori yang telah ditambahkan.

Halaman Post

Halaman post pada admin panel, berisikan post berita yang telah ditambahkan.

Halaman Tambah Post

Halaman tambah post untuk menambahkan post berita dengan memasukkan judul berita, gambar berita, isi berita, dan memilih kategori berita sesuai dengan kategori yang telah ditambahkan.

Halaman Feedback

Halaman feedback, menampilkan feedback yang diberikan oleh pengguna.

Halaman Home

Company Profile sederhana menggunakan tailwind css, berisikan halaman home, about, layanan, dan kontak.

Halaman About

Halaman ini menampilkan deskripsi singkat dari company.

Halaman Layanan

Halaman ini menampilkan layanan yang ditawarkan oleh company.

Halaman Kontak

Halaman ini menampilkan kontak dari company.

Halaman Landing Page

Halaman landing page ketika masuk ke sistem.

Halaman Dashboard

Halaman dashboard berisikan jumlah mahasiswa yang terdaftar dan berisikan grafik data dari hasil perhitungan.

Halaman Dashboard

Dari halaman dashboard ini terdapat grafik presentase dari perhitungan poin terbanyak dari kategori yang ada, serta ranking peserta 10 skor terbesar.

Halaman Tambah

Halaman ini berfungsi untuk menambahkan peserta yang akan dilakukan perhitungan.

Halaman Ranking

Halaman ini menunjukkan ranking seluruh peserta.

Halaman Detail

Halaman detail peserta ini menampilkan hasil perhitungan detail dari peserta.
